# Task Manager API App

A simple full-stack task manager application that allows users to create, view, update, and delete tasks.
The project is built using a FastAPI backend, a React frontend, and Docker for containerization.

## What Is It

- Add a task with a title and description
- View a list of all tasks
- Update existing tasks
- Delete tasks

Note: The frontend communicates with the backend through a REST API.

## Technologies Used

- **Backend:** FastAPI (Python)
- **Frontend:** React (JavaScript)
- **Containerization:** Docker & Docker Compose
